# 1. HTML?
* HTML (Hyper Text Markup Language)의 약자 입니다.
* PAGE의 구조를 설명
* html은 아래와 같이 태그(<, >)라는 기호로 구성이 되며, 닫을 때에는 슬래쉬(/)를 붙여주어야 한다.


```HTML

<html>
    <head>
        <title> 타이틀입니다. </title>
    </head>
    <body>
        <h1> 제목 크기1 </h1>
        <p> 단락1 (paragraph) </p>
        <h2> 제목 크기2 </h2>
        <p> 단락2 (paragraph) </p>
    </body>
</html>

<!--  * html에서의 주석은 이렇게 표현할 수 있습니다. -->

```

<br>

# 2. HTML의 간략한 구조

* html은, 꺾쇠 괄호의 태그를 통해서 표현합니다.
* html, head, body 등의 요소를 통해서 표현할 수 있습니다.

```
<HTML> → 시작 태그를 의미합니다.
</HTML> → 종료 태그를 의미합니다.
```
